# Webpack-starter
Este es un proyecto de una aplicaciones web de Calculadora utilizando webpack.
### Notas:
Recuerde reconstruir los modulos de Node JS
....
npm install
....
Y para reconstruir la carpeta dist
....
npm run build
....
Para el ambiente de desarrollo
....
npm run build:dev
....
